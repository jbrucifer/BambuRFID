Clone Tag

Cloning copies all data including the RSA signature from a genuine Bambu Lab tag to a Magic/FUID MIFARE Classic tag. The destination tag must support UID changes (Gen2 or FUID tags). Standard tags have read-only UIDs and cannot be cloned to.
